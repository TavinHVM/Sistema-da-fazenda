Manual de Uso do Sistema da Fazenda

Introdução

O Sistema da Fazenda foi desenvolvido para gerenciar de forma eficiente as operações de registro e controle de clientes, funcionários, fornecedores e vendas de uma fazenda. Este manual tem como objetivo orientar o usuário sobre o uso das funcionalidades do sistema, incluindo o acesso ao sistema, cadastro de informações e controle de permissões baseado no tipo de usuário.

Objetivos do Sistema

  • Cadastro de clientes, funcionários e fornecedores.
  • Registro de vendas e produtos.
  • Controle de acesso baseado em login (admin ou funcionário).
  • Permissões diferenciadas para cada tipo de usuário.

Pré-requisitos de Uso

  • Dispositivo com acesso à internet e navegador atualizado.
  • Credenciais de acesso ao sistema.

Funcionalidades do Sistema

1. Tela de Login

Ao acessar o sistema, a primeira tela apresentada é a de login, onde o usuário deve inserir suas credenciais para acessar as funcionalidades correspondentes ao seu perfil.

  1. Insira o login no campo "Login".
  2. Insira a senha no campo "Senha".
  3. Clique no botão "Login".

Dependendo das credenciais fornecidas, o usuário será redirecionado para a tela inicial com permissões específicas:

  • Admin: acesso completo ao sistema.
  • Funcionário: acesso restrito, sem permissão para acessar o cadastro de funcionários.

2. Tela Inicial

Após o login, o usuário é direcionado para a tela inicial, que apresenta as seguintes opções:

  • Cadastros: permite o acesso aos módulos de cadastro de clientes, funcionários e fornecedores.
  • Registro de Vendas e Produtos: direciona para as telas de registro de vendas e controle de estoque.
  • Feedback: (futuro módulo para controle de satisfação).
  • Manual de Uso: acessa este manual de instruções.
  • Sair: desloga o usuário e retorna à tela de login.

3. Tela de Cadastros

Na seção de cadastros, o usuário pode gerenciar informações de clientes, funcionários e fornecedores. As permissões variam conforme o perfil:

  • Admin: acesso total a todas as opções de cadastro.
  • Funcionário: acesso restrito, sem permissão para cadastrar funcionários.

Para realizar um cadastro:

  1. Clique no botão correspondente ao cadastro desejado (Clientes, Funcionários ou Fornecedores).
  2. Preencha os campos obrigatórios com as informações solicitadas.
  3. Clique em "Gravar" para salvar ou "Limpar" para resetar os campos.

4. Tela de Registro de Vendas

Esta tela permite o registro e controle de vendas realizadas. Para registrar uma venda:

  1. Clique em "Registro de Vendas e Produtos" na tela inicial.
  2. Preencha os dados da venda, como cliente, funcionário responsável, produto, quantidade e valor.
  3. Clique em "Gravar" para salvar a venda.

5. Controle de Permissões

O sistema possui controle de permissões baseado no tipo de usuário:

  • Admin: pode acessar todas as telas do sistema e realizar todas as operações.
  • Funcionário: possui restrições, como:
    • Não pode acessar a tela de cadastro de funcionários.
    • Não pode realizar operações administrativas.

Conclusão

O Sistema da Fazenda foi desenvolvido para otimizar as operações administrativas e comerciais, garantindo controle e eficiência. Com funcionalidades acessíveis e diferenciadas para administradores e funcionários, o sistema é uma ferramenta robusta para atender às necessidades de gerenciamento.
